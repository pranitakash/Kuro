# Kuro Music App

Kuro is a clean, modern Android music streaming application built with Jetpack Compose and Media3. It allows users to stream music from open sources with a premium, monochromatic UI.

## Features

- **High-Fidelity UI**: Monochromatic design with smooth animations.
- **Fast Playback**: Optimized stream resolution and aggressive buffering for instant music.
- **Lyrics Support**: Synchronized lyrics via LRCLIB.
- **Offline Support**: Cache and download songs for offline listening.
- **Monochromatic Aesthetics**: Rotating vinyl disc and waveform seek bars.

## Tech Stack

- **UI**: Jetpack Compose + Material 3
- **Playback**: Media3 (ExoPlayer)
- **Networking**: Retrofit + OkHttp
- **Database**: Room
- **DI**: Hilt

## Getting Started

1. Clone the repository.
2. Open in Android Studio (Iguana or newer).
3. Build and run on an Android 8.0+ device.
